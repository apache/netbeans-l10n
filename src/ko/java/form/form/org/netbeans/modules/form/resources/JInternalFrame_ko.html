MDI 사용자 인터페이스를 제공하는 JDesktopPane 상에 위치할 수 있는 JFC (Swing) 내부 프레임을 만듭니다. 일반적으로 각 개별 프레임을 위한 디스플레이 엘리먼트를 포함하고 있는 내부 프레임 클래스를 만들 수 있습니다. JFrame 클래스와 같은 각 내부 프레임 클래스를 다룹니다. JDesktopPane 을 가진 JFrame (또는 JApplet) 응용 프로그램을 만듭니다. 코드를 사용하거나, 혹은 내부 프레임이 컴파일된 후 프로젝트 창의 복사하여 붙여넣기 기능을 사용하여 데스크톱 패인에 내부 프레임을 추가합니다.
