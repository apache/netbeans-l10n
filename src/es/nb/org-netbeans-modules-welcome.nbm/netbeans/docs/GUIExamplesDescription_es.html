Descripciones de Ejemplos de Formularios de Interfaz de Usuario

Este proyecto contiene tres formularios visuales que fueron diseñados con el Constructor de Interfaz de Usuario del IDE.

Para abrir un formulario de ejemplo en el constructor de GUI :

  1. En la ventana proyectos de IDE's , navegando en el nodo de paquetes predeterminado sin el
    Directorio de Paquetes Fuentes.

  2. Doble-click el nodo del formulario de ejemplo que busca ver.

    El IDE larga el Constructor de Interfaz de Usuarioy muestra el formulario seleccionado.


Editor de Contacto

Este formulario complejo es similar diálogos de Windows usados para mostrar y editar la información de un base de datos. El ejemplo ilustra todas las caracteristicas destacadas del Constructor de Interfaz de Usuario, incluyendo alineación, anclaje, redimensionado, tabulación, comportamiento de auto redimensionado en tiempo de ejecución. Para ver un tutorial detallado que incluye demostración flash mostrando como construir este formulario, vea Diseñando un GUI Swing en el IDE NetBeans .


Preferencias de Antena

Este ejemplo ilustra a tabla basada en formularios comunmente usadas para mostrar información en grupos visualmente coherentes. La funcionalidad de alineación del Constructor de Interfaz de Usuario le permite agrupar y redimensionar componentes no adjacentes (como labels, campos de texto, y botones) consistentemente sin la necesidad de fusionar las celdas manualmente.


Encontrar

Este simple formulario ilustra diseño utilizado a menudo en varias aplicaciones que requieren una funcionalidad básica de búsqueda. Las funcionalidades de alineamiento y redimensionado del Constructor de Interfaz de Usuario le permiten facilmente la construcción de esos formularios, garantizando al mismo tiempo que localización predecible.
