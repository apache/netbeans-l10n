Crea l'esquelet d'un servei web a partir d'un fitxer WSDL. Els serveis web són components de programari re-utilitzables que encapsulen semànticament funcionalitats discretes. Els serveis web són accessibles sobre protocols estàndard com per exemple SOAP. Els serveis web creats amb aquesta plantilla es despleguen i s'executen d'acord amb els estàndards de la JSR-109 i del J2EE 1.4.
