Description de l'exemple de formulaire de l''interface graphique

Ce projet contient trois formulaires visuels qui sont conçus avec le générateur d''interface graphique.

Pour ouvrir un exemple de formulaire dans le générateur d''interface graphique :

  1. Dans la fenêtre Projets de l''IDE, naviguer vers le noeud de paquetage par défaut à l'intérieur de
    Répertoire des paquetages de source.

  2. Double cliquer sur le noeud de l''exemple de formulaire que vous voulez voir.

    L''IDE lance le générateur d''interface graphique et affiche le formulaire choisi.


Éditeur de contact

Ce formulaire complexe ressemble aux boites de dialogue utilisées pour afficher et éditer l''information d''une base de données. Ce exemple illustre la large gamme de fonctionnalités du générateur d''interface graphique comme l''alignement, l''ancrage, le dimensionnement, l''indexation et l''auto-dimensionnement du runtime. Pour regarder un tutoriel détaillé incluant les démonstrations flash qui montre comment concevoir ce formulaire, voir Concevoir un interface graphique utilisateur Swing dans l''EDI Netbeans .


Antenne Préférences

Cet exemple illustre un type de formulaire basé table généralement utilisé pour afficher une information relative à des groupes visuellement cohérent. Les fonctionnalités avancées du générateur d''interface graphique vous permettent de grouper et dimensionner les composants non-adjacents ( tels que les étiquettes, les zones de textes et les boutons) toujours sans avoir à fusionner les cellules manuellement.


Rechercher

Ce formulaire simple illustre une mise en place simple souvent utilisée dans diverses applications nécessitant une fonctionnalité de recherche basique. Le fonctionnalités de dimensionnement et d''alignement du générateur d''interface graphique vous permettent de construire rapidement de tels formulaires en assurant qu''on traduise de façon prévisible.
