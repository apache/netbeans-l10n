Module non activé

La rubrique d'aide que vous essayez de visualiser est située dans le module {0}, qui est actuellement désactivé. Pour activer des modules installés désactivés, procédez comme suit :

  1. Sélectionnez Outils > Modules d'extension (Préférences NetBeans sur Mac).
  2. Accédez à l'onglet Installé.
  3. Sélectionnez le module {0} et cliquez sur Activer.
