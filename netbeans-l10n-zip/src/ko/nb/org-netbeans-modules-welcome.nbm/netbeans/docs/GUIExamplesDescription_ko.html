GUI 폼 예제 설명

이 프로젝트는 IDE의 GUI 빌더로 설계된 3개의 비주얼 폼을 포함하고 있습니다.

GUI 빌더에 예제 폼을 엽니다 :

  1. IDE 프로젝트 창에서, 기본 패키지 노드를 탐색하십시오.
    소스 패키지 디렉터리

  2. 보고자 하는 샘플 폼의 노드를 더블 클릭하십시오.

    IDE는 GUI 빌더를 런칭하고 선택된 폼을 표시합니다.


콘택트 편집기

이 컴플렉스 폼은 일반적으로 데이터베이스 정보를 표시하고 편집하는데 사용되는 윈도우 다이얼로그와 유사합니다. 예제는 정렬,고정,크기 조정,들여쓰기,런타임시 자동-크기 조정 동작을 포함하는, GUI 빌더 기능의 전체 범위를 보여줍니다. 이 폼을 어떻게 빌드하는 지를 보여주는 플래시 데모를 포함한 자세한 튜토리얼을 참조하려면, 넷빈즈 IDE의 스윙 GUI 설계하기 를 보시기 바랍니다.


Antenna 환경설정

이 예제는 일반적으로 시각적으로 일관된 그룹 관련 정보를 보여주는데 사용되는 테이블 기반 폼 양식을 보여줍니다. GUI 빌더의 향상된 정렬 기능은 셀을 수동으로 병합할 필요없이 인접하지 않은 컴포넌트를 (예를 들어, 라벨, 텍스트필드, 및 버튼) 일관되게 그룹화하거나 크기 조정을 가능하게 합니다.


찾기

이 심플 폼은 종종 기본 탐색 기능을 요구하는 다양한 응용 프로그램에서 사용되는 직관적 레이아웃을 보여줍니다. GUI 빌더의 정렬 및 크기 조정 기능은 로컬화 기능을 예측 가능하도록 강화하여 폼을 보다 신속하게 구축할 수 있습니다.
