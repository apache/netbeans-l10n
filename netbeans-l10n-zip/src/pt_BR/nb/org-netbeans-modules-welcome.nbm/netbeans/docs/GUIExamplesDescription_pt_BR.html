Descrições de exemplos de forms de GUI

Este projeto contém três forms visuais criados com o Construtor de GUI do IDE.

Para abrir um form de exemplo no construtor de GUI:

  1. Na janela Projetos do IDE, vá até o nó do pacote default no
    diretório Pacotes de código-fonte.

  2. Clique duas vezes no nó do form de amostra que deseja visualizar

    O IDE aciona o construtor de GUI e exibe o form escolhido.


Editor de contato

Este form complexo é semelhante às caixas de diálogo do Windows normalmente usadas para exibir e editar informações de bancos de dados. O exemplo ilustra toda a série de funcionalidades do construtor de GUI, incluindo alinhamento, ancoramento, dimensionamento, recuo e comportamento de redimensionamento automático em runtime. Para visualizar um tutorial detalhado incluindo demonstrações em flash que mostram como construir este form, consulte Construindo Swing GUI no NetBeans IDE.


Preferências de Antena

Este exemplo ilustra um tipo de form baseado em tabela usado normalmente para exibir informações relacionadas em grupos visualmente lógicos. As funcionalidades avançadas de alinhamento do construtor de GUI permitem agrupar e dimensionar os componentes não adjacentes (como labels, campos de texto e botões) de forma coerente sem ter que mesclar as células manualmente.


Localizar

Este form simples ilustra um layout comum usado frequentemente em várias aplicações que requerem a funcionalidade de pesquisa básica. As funcionalidades de alinhamento e dimensionamento do construtor de GUI permitem construir tais forms rapidamente e garantir que eles estejam bem situados.
