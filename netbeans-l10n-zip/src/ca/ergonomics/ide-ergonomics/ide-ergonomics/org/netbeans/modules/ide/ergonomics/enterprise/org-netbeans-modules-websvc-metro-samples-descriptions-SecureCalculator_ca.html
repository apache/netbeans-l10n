Política de seguretat que condueix un servei i client de servei web d'una calculadora desenvolupats mitjançant la pila de serveis web Metro. Conté tant el client com el servidor, el client necessita autentificar-se al servei mitjanant l'autentificació de nom d'usuari.
Nota: l'exemple necessita que es registri una instància del servidor d'aplicacions Java EE 6 (GlassFish V3).
