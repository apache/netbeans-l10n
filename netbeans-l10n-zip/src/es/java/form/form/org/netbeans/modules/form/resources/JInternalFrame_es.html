Crea una nueva ventana interna JFC (Swing) que se puede colocar sobre un JDesktopPane para proporcionar una interfaz de usuario MDI. Normalmente, se creará una clase de tipo ventana interna que contenga elementos visuales para cada ventana individual. Trate todas las clases de ventanas internas como una clase JFrame. Después cree una aplicación JFrame (o JApplet) conteniendo un JDesktopPane. Añada las ventanas internas al panel de escritorio utilizando código o utilizando las funciones Copiar y Pegar en la ventana Proyectos, después de que se compilen las ventanas internas.
